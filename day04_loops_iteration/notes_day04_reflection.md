# ✍️ Day 4 Reflection

## Apa yang saya pelajari hari ini
Saya belajar bagaimana sistem automation dapat mengulangi tugas menggunakan loop.
Konsep “loop” ini membuat Python bisa bekerja seperti mesin otomatis — terus bekerja sampai selesai.

## Tantangan saya
Kadang bingung kapan sebaiknya pakai `for` dan kapan `while`.
Saya akan latihan membuat kondisi berhenti (`break`).

## Hubungan ke dunia nyata
Dalam automation, banyak proses berbasis loop:
- Loop pelanggan untuk kirim laporan,
- Loop data harian untuk validasi,
- Loop cek status API.
Tanpa loop, automation tidak mungkin jalan terus secara mandiri.
