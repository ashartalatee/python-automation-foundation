# ✍️ Day 2 Reflection

## Apa yang saya pelajari hari ini
Saya memahami bahwa setiap data memiliki tipe yang menentukan bagaimana Python memprosesnya. Ini penting karena automation bergantung pada keakuratan data dan cara mengolahnya.

## Apa tantangan saya
Masih perlu membiasakan diri untuk berpikir “data-driven”, bukan hanya “code-first”.

## Bagaimana ini terhubung ke automation
Dalam automation, setiap data harus bisa dibaca, diubah, dan dikirim ke sistem lain. Variabel adalah penghubung antar proses tersebut.
