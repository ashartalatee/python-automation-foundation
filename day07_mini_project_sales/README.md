# Day 07 — Mini Project: Analisis Penjualan Harian

Pada Day 07, saya menggabungkan semua dasar Python (day 1–6) menjadi satu mini pipeline menggunakan **MALS (Modular Automation Learning Structure)**.

## Struktur MALS
- **Input Module** → Mengambil data penjualan
- **Processing Module** → Membersihkan dan memvalidasi data
- **Analysis Module** → Menghitung metrik utama
- **Output Module** → Menampilkan laporan
- **run_pipeline.py** → Pipeline utama yang mengorkestrasi semuanya

## Kenapa ini penting?
- Melatih pola pikir automation architect
- Belajar membangun sistem modular sejak awal
- Mempersiapkan pondasi DataOps & Automation
